to start a screen:
> screen -S <nameOfScreen>

Then start a process, e.g.
> bash run.sh

Detach the screen by pressing Ctrl+A, Ctrl+D

To see which screens are running:
> screen -ls

To reconnect to a screen:
> screen -r <nameOfScreen>
